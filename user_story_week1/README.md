# Sales Cost Calculator

## Description

This project is a simple Python program that calculates the total cost of a product purchase. The user enters the product name, price, and quantity, and the program shows the total cost.

## How it works

The program asks the user to enter the name of the product, the price, and the quantity. Then it multiplies the price by the quantity to calculate the total cost. Finally, it displays the information and the total result on the screen.
